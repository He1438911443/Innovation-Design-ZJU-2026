## 问题 1：水晶在视角中太大

**现象**：相机距离晶体太近，画面被 3-4 颗大晶体占满，没有洞穴空间感。

**原因**：`final_scene.py` 的 `_build_immersive_camera` 函数中 `dist = 3.5`（相机距晶体只有 3.5m），且球壳半径 22（晶体紧贴墙壁），相机没有退后空间。

**修改** (`final_scene.py`):
- 找到 `_compute_camera_keyframes` 函数（大约 line 380）
- 把 `dist = 3.5` 改成 `dist = 8.0`
- 把 `radius=22` 改成 `radius=28`（在 `_build_cave_shell` 函数中，大约 line 308）
- 把 `cmds.move(0, 5, 0, shell)` 改成 `cmds.move(0, 10, 0, shell)`
- 把 Y scale `0.55` 改成 `0.30`

## 问题 2：没有光影效果 — 洞穴一片灰暗

**现象**：场景整体偏暗且缺乏对比度，没有「暖金主光从入口照进来 + 晶体发出彩色微光 + 远处被雾吞没」的层次感。

**原因**：
1. `aiSkyDomeLight` 环境光强度太低 (0.65) 但均匀照亮全场，抹平了对比度
2. 入口 `directionalLight` 强度 120 但没有正确指向晶体群
3. 晶体 glow 点光源 `decayRate=0`（无衰减）导致光到处漫射
4. 体积雾 `density=0.008` 太淡，没有丁达尔效果

**修改** (`final_scene.py`):

### 灯光部分（大约 line 690-710）：
```python
# Dome: 降一半，让洞穴暗部更深
_safe_attr(dome_lt, "intensity", 0.30)  # 原来是 0.65

# Key light: 暖金色，从洞穴入口射入
_safe_attr(key, "color", (1.0, 0.75, 0.40), "color")  # 更暖
_safe_attr(key, "intensity", 180)  # 原来 120，提高对比度
cmds.rotate(-55, -35, 0, key)  # 更陡的角度，射入更深
cmds.move(20, 18, 25, key)  # 从洞穴入口上方射入

# Rim light: 冷蓝色背光勾勒晶体轮廓
_safe_attr(rim, "color", (0.15, 0.30, 1.0), "color")
_safe_attr(rim, "intensity", 40)  # 原来 25

# Crystal glows: 让衰减生效，光只在晶体周围亮
for i, (pos, color, *_) in enumerate(anchors[:20]):
    lt = cmds.pointLight(name="CCV9_glow_{:02d}".format(i))
    _safe_attr(lt, "color", color, "color")
    _safe_attr(lt, "intensity", random.uniform(2000, 4000))  # 2-4x 原来
    _safe_attr(lt, "decayRate", 1)  # 线性衰减！原来=0 无衰减
```

### 体积雾（大约 line 720）：
```python
_safe_attr(fog, "density", 0.02)  # 2.5x 原来0.008，可见丁达尔光束
```

### 相机曝光（大约 line 575）：
```python
try: cmds.setAttr(cam + "Shape.ai_exposure", 1.5)  # 原来 3.0，太亮
except: pass
```

## 问题 3：没有次表面散射的视觉通透感

**现象**：晶体看起来是实心的彩色玻璃，没有「光从内部透出来」的 SSS 效果。

**原因**：
1. `subsurfaceWeight` 只有 0.75，`subsurfaceRadius` 太小 (0.45)，光线穿透不深
2. `transmissionWeight` 只有 0.58，晶体不够透
3. 没有用 Arnold 的 `subsurface` color 做彩色散射

**修改** (`final_scene.py` 的 `_material` 函数，大约 line 44-50)：
```python
_safe_attr(shader, "subsurfaceWeight", 0.95)   # 原来 0.75 — 更强 SSS
_safe_attr(shader, "subsurfaceRadius", (0.8, 0.8, 0.8), "color")  # 更深穿透
_safe_attr(shader, "subsurfaceScale", 4.0)     # 原来 2.8 — 更大散射半径
_safe_attr(shader, "transmissionWeight", 0.72) # 原来 0.58 — 更透光
_safe_attr(shader, "transmissionDepth", 5.0)   # 原来 3.0 — 更深穿透
```

## 问题 4：尘埃太大太亮

**现象**：120 颗尘埃粒子半径 0.012-0.04，是亮的橙色球体，不像悬浮微尘。

**修改** (`final_scene.py`，大约 line 550)：

# Dust: smaller, more transparent
dust_mat = _material("CCV9_dust", (0.9, 0.85, 0.75), emission=.08, roughness=.35)

    ...

for i in range(200):  # 原来 120

## 问题 5：相机漫游速度太快

**现象**：动画播放时帧与帧之间跳动太明显。

**修改** (`final_scene.py` 的 `_compute_camera_keyframes`):
```python
# 原来是 9 个关键帧分布在 960 帧上
# 改成 30 个关键帧分布在 2400 帧上（100秒 at 24fps）
total_frames = 2400
num_keyframes = 30
```

同时在 `_build_immersive_camera` 末尾：
```python
cmds.playbackOptions(min=0, max=2400, animationStartTime=0, animationEndTime=2400)
```

## 问题 6：地形太浅，没有「洞穴深处」

**现象**：terrain bowl 不够深，相机一飞就看到整个洞穴全貌，没有探索感。

**修改** (`_build_diamond_square_terrain`，大约 line 270)：
```python
# 原来
cy = hm[i][j] * (1.0 - dist) - 16 * dist ** 1.5 + math.sin(cx*0.35)*math.cos(cz*0.35)*2.5

# 改成 — 更深的地形碗
cy = hm[i][j] * (1.0 - dist) - 22 * dist ** 1.5 + math.sin(cx*0.4)*math.cos(cz*0.4)*3.0
```
