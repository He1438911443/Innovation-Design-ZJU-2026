## 2. 课程主要内容

### 模块一：Python 基础 + Maya 脚本环境（Day 1-2）

- Maya Script Editor 界面、Python Tab 切换
- Hello World → 字符串（引号、连接、转义 `\n`、三引号 `"""`）
- `ord()` / `chr()`、`type()` 函数
- ⭐ 老师亲自演示 Maya 的 VS Code + AI 插件联调

### 模块二：Python 数据结构 + 控制流（Day 3 上午）

- List 操作：`append`、`remove`、`sort()` vs `sorted()`、索引、合并、`pop`
- While 循环、For 循环 + `range()`
- If/elif/else 条件分支
- 变量作用域、字符串格式化 `f"...{score}..."`

### 模块三：Maya 节点系统 + 动画入门（Day 3 上午）

- Maya Node 概念（mesh node、transform node、shader node）
- 属性读写：`cmds.getAttr()`、`cmds.setAttr()`
- 关键帧动画：`cmds.setKeyframe()`、`cmds.currentTime()`
- ⭐ 老师演示从零到动画的完整 Python 代码

### 模块四：各组预汇报 + 城市设计讨论（Day 4 下午 最关键的课）

各组展示初步设计方案，老师现场点评：

| 组 | 方向 | 老师点评 |
|----|------|---------|
| Group 5 | 三层地形 + 浮空城市 | "Next time try English" |
| Group 6 | 不同功能建筑类型 | "very interesting" |
| Group 7 | Zombie风格 | "infection points spread out, ruined city, walls closing up... idea is good but implementation challenging" |
| Group 8 | 网格布局 + 距离控制高度 | "add variation in details. Good idea" |
| Group 9 | 神经网络结构建模 | "very challenging. The picture from AI looks impressive" |

### 模块五：Python + Maya 进阶（Day 4 下午后半段）

- 嵌套列表（matrix）、2D 列表遍历
- 函数定义 `def function():`
- 代码结构规范：import → 全局变量 → 函数定义 → 主入口
- ⭐⭐ **地形生成教程**：老师现场打开 Maya 展示地形（terrain object → terrainMesh → 在面上选随机点 → 生成建筑）

### 模块六：AI 伦理 + 行业讨论（Day 4 末段）

- 好莱坞编剧罢工（148 天）：AI vs 人类创意
- Opt-in vs Opt-out 版权模型
- 中国 AI 微短剧现状
- Prof Yang 本人的研究：采访 200 位电影从业者

## 6. 技术栈说明

### Maya 是什么
Autodesk Maya 是电影/游戏行业标准的 3D 动画软件。这个课里你不是手动建模——而是**写 Python 代码让 Maya 自动生成场景**。

### 技术栈
| 技术 | 做什么 | 你需要什么程度 |
|------|--------|---------------|
| Python | 编程语言 | 基础语法+函数+列表，AI辅助完全够 |
| Maya | 3D 软件 | 打开 Script Editor，粘贴运行代码 |
| Maya Python API (`maya.cmds`) | Maya内置Python命令 | `polyCube()`, `move()`, `group()`, `setAttr()` 等 |
| VS Code + Maya插件 | AI辅助编程 | Prof Yang 强烈推荐 |
